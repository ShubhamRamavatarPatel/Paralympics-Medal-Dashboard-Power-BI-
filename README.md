📊 Paralympics Medal Performance Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Paralympics Medal Performance Dashboard developed using Microsoft Power BI.
The dashboard provides insights into medal distribution, country performance, and historical trends in Paralympic Games.

It allows users to explore medal statistics through interactive visuals, KPIs, and slicers.

🎯 Objectives

Analyze Paralympic medal distribution across countries.

Track medal trends across different years and seasons.

Identify top-performing countries.

Provide interactive filtering for better data exploration.

📊 Dashboard Features
KPI Indicators

The dashboard includes key performance indicators:

Total Medals – 29,338

Gold Medals – 10,832

Silver Medals – 9,472

Bronze Medals – 9,034

Total Countries Participated – 20

📈 Visualizations
1️⃣ Medal Breakdown by Country

Chart Type: Stacked Column Chart

Displays medal composition (Gold, Silver, Bronze) for top countries.

2️⃣ Medal Trend by Season Over Time

Chart Type: Line Chart

Shows how total medals changed across years for Summer and Winter Paralympics.

3️⃣ Top Countries by Total Medals

Chart Type: Bar Chart

Highlights the countries with the highest medal counts.

4️⃣ Total Medals by NPC Rank

Chart Type: Column Chart

Displays medal distribution based on NPC ranking.

🎛 Interactive Filters

The dashboard includes slicers for dynamic filtering:

Year – Filter data by Paralympic year.

Season – Switch between Summer and Winter Games.

Additionally, a Clear All Slicers button allows users to quickly reset filters.
